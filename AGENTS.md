## Development

When starting the dev server, use background mode:

```
astro dev --background
```

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.

## Design direction
- Desktop canvas metaphor: folders scattered on a neutral background
- Click a folder → opens a Finder-style window overlay
- Folders: Work/Projects, About me, Play/Side projects, Writing/Blog, Contact
- Mini-me photo cutout characters (realistic, Image 6 style) sitting on/near folders
- Colour palette: cool minimal (white, grey, blue)
- Stack: Astro + vanilla CSS, no framework components yet
